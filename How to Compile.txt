- Open this folder in VSCode
- Open VSCode terminal and put this command to compile everything:

	g++ *.cpp projectile/*.cpp enemies/*.cpp towers/*.cpp -IC:\mingw_dev_lib\include\SDL2 -LC:\mingw_dev_lib\lib -w -lmingw32 -lSDL2main -lSDL2 -lSDL2_image -lSDL2_mixer -lSDL2_ttf

- Run the game by this command:

	.\a.exe

- To transpile to html,js:

	em++ bombtower.cpp enemy.cpp firetower.cpp game.cpp gameManager.cpp goldtower.cpp highHPZombie.cpp highSpeedZombie.cpp icetower.cpp longbowtower.cpp main.cpp normalZombie.cpp object.cpp repairtower.cpp specialZombie.cpp tower.cpp weakZombie.cpp projectile.cpp -o main.html -s USE_SDL=2 -s USE_SDL_IMAGE=2 -s SDL2_IMAGE_FORMATS='["png"]' -s USE_SDL_MIXER=2 --preload-file assets --bind --use-preload-plugins
